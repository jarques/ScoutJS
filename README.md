<h1>Scout.js - Super simple Geolocation</h1>

Simply call scout.get() to get the user's location.

Example:

<pre>
scout.get(function(coords){
  alert(coords.latitude + ", " + coords.longitude); 
});
</pre>

You can view an example here: http://jarqu.es/scout/
